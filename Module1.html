Датчик для избегания препятствий KY-032
“KY-032” – модуль представляет собой инфракрасный датчик обнаружения препятствий. Устройство имеет передатчик (IR LED), излучающий инфракрасный луч частотой 38 кГц, который принимается приемником (IR receiver) на плате. При приближении какого-либо предмета к сенсору (близость регулируется потенциометром) на выходе “OUT” появляется низкий уровень напряжения и включается светодиод на плате модуля.
1. digitalWrite( enablePin, HIGH); //Включить внутренний сигнал 38 кГц
2. microDelay( 210); //Задержка 210µs(8 пульсов по 38kHz)
3. if( digitalRead( outputPin)) //Если Output детектора равен HIGH
4. {
5. objectDetect = false; //Препятсвия не было найдено
6. }
7. else //Но если Output равен LOW,
8. {
9. microDelay( 395); //Опять ждать 15 пульсов
10. if( digitalRead( outputPin)) //Если Output теперь равен HIGH
11. {
12. objectDetect = false; //Препятсвия не было найдено
13. }
14. else //Если Output до сих пор LOW
15. {
16. objectDetect = true; //Препятсвие было обнаружено
17. }
18. }
19. digitalWrite( enablePin, LOW); //Выключить внутренний сигнал 38kHz
